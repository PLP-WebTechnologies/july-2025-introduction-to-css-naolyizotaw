Understanding CSS and the Box Model

This page demonstrates basic CSS styling and the fundamental concepts of the CSS Box Model. Each element on this page is a box consisting of content, padding, borders, and margins.

Content Box

The inner part of the box where your text and images appear. Its dimensions can be set with width and height properties.

Padding

Clears an area around the content. The padding is transparent and affects the element's background.

Border

A border that goes around the padding and content. You can style it with different colors, widths, and styles.

Margin

Clears an area outside the border. The margin is completely transparent and creates space between elements.
